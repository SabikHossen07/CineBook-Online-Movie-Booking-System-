# 🎬 CineBook - Movie Ticket Booking System

CineBook is a full-stack movie ticket booking system built using PHP, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features

- User Registration & Login
- Movie Listing & Details
- Showtime Browsing
- Seat Selection System 🎟️
- Booking Confirmation
- E-Ticket Generation (PDF)
- My Bookings Page
- Cancel Booking Option
- Manager Panel (Manage showtimes & seats)

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (Local Server)

---

## 💻 How to Run Locally

1. Install XAMPP
2. Start Apache & MySQL
3. Copy project folder into:

C:\xampp\htdocs\cinebook

4. Open phpMyAdmin
5. Create database:

cinebook

6. Import `cinebook.sql`
7. Open browser:

http://localhost/cinebook/
